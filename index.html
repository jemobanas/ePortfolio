Project 1 Title: Pet Adoption Website

This project aimed to create a user-friendly website to help connect people with adoptable pets. Technologies Used: HTML CSS JavaScript PHP MySQL Key Features: Pet Listing and Browsing Pet Adoption Application Learning Outcomes: Gained proficiency in front-end development with HTML, CSS, and JavaScript to create an interactive and responsive user interface. Learned to integrate a back-end database (MySQL) with a website using PHP to store and retrieve pet information dynamically. Enhanced understanding of website development processes, including planning, designing, coding, and testing.

Project Title: K-Means Clustering Visualization and Evaluation

This project implements the K-Means clustering algorithm to group unlabeled data into clusters based on their similarity. The code generates synthetic data, visualizes the iterative process of cluster formation, and evaluates the clustering performance using metrics like the Silhouette Score, Davies-Bouldin Index, and Adjusted Rand Index.

Project Title: Turing Machine Simulator

Project Description:This Turing Machine simulator, built with Tkinter, allows users to input strings and test them against predefined Turing Machine rules to visualize their execution and determine acceptance. This project was developed for an automata theory course to provide a practical understanding of Turing machine concepts.
